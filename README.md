# Tweet Sentiment Classification using NLP Embeddings

## Overview
This project performs sentiment analysis on tweets using Natural Language Processing (NLP) techniques.  
The goal is to classify tweets into different sentiment categories such as positive, negative, or neutral.

This project was developed as part of learning machine learning and NLP concepts using Python.

## Features
- Text preprocessing and cleaning
- NLP embeddings for feature representation
- Sentiment classification model
- Evaluation of model performance

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Natural Language Processing (NLP)

## Dataset
The dataset used for this project is taken from Kaggle and contains labeled tweets with sentiment categories.
